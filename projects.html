Learning Rails in 2020

Shoppingfy is a naive implementation of Shopify. The app has been deployed with Heroku here.

This app was built with Ruby on Rails 6, and the front end uses raw css and bootstrap

Testing environment: Minitest

Shoot-io is multiplayer top-down shooter game developed with Phaser, a gaming framework made by Photon Storm. The game has been deployed with Heroku and is playable.

Front end: Phaser, React

Back End: Node.js, Express, Socket.io, PostgreSQL

Scheduler is a single page web application that allows users to book, edit, and delete interviews. The schedule is served by a server running separately and interviews that are booked on the client side will update the server and automatically render the new appointment.

Front end: HTML, CSS, react

Back end: Node.js, Express, PostgreSQL, Axios

Testing environment: Jest, Storybook, Cypress

Tweeter is a single page replica of twitter that allows users to post tweets.

Front end: HTML, CSS, JQuery, Bootstrap, SASS

Back end: Node.js, Express, Ajax

Quizo is an application that allows you to make quizzes for any topic.

Front end: HTML, EJS, CSS, JQuery, Bootstrap

Back end: Node.js, Express, PostgreSQL, Ajax
